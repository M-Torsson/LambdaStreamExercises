# Lambda Stream Exercises

This project contains exercises about functional programming and the Stream API in Java.

## What I did

- Created a functional interface (PersonRule)
- Used lambda expressions to define rules (active, adult, lives in Stockholm)
- Implemented a method to filter people based on rules
- Created another functional interface (PersonAction)
- Applied actions like printing names and sending emails

## Stream API

- Filtered active people
- Mapped names
- Counted adults
- Sorted people by age
- Found first active person in Stockholm

## Extra
